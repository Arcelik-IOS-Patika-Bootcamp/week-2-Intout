## Notepad Application.

Application contains 3 views:
- Slider View
- Main Canvas
- Text Editor

Main Canvas is the parent view of the other views and it contains color edit objects. When color changes this information will be directly updated on Slider View class. Color information on Text Editor will be updated with a delegate.
